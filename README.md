## Features

- Real-time messaging  
- Responsive design for desktop and mobile  
- Simple and intuitive interface  
- Hosted live on Firebase  

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Hosting:** Firebase (Firestore, Hosting)  
- **Authentication:** Firebase Auth (optional)

## Usage

To use ChitChat:

1. Open the [live app](https://chitchat-254bc.web.app/) or run it locally.  
2. If authentication is enabled, sign in or create a new account.  
3. Start chatting in real-time with other users.  
4. Enjoy the responsive interface on both desktop and mobile devices.
