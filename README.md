ChitChat

Live Demo

ChitChat is a real-time chat application built with modern web technologies. It allows users to connect, send messages instantly, and enjoy a simple, intuitive interface for online communication.

Features

Real-time messaging

User-friendly interface

Responsive design for desktop and mobile

Modern and lightweight front-end

Hosted live on Firebase

Tech Stack

Frontend: HTML, CSS, JavaScript

Backend & Hosting: Firebase (Realtime Database / Firestore, Hosting)

Other: Firebase Authentication (if implemented)
